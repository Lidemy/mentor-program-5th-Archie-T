## 交作業流程

1.   git clone 遠端資料到本機 ： git clone ( 網址 )
2.   在本機的  master 上開 branch： git branch -b *( 版本 )*
3.   於 branch 的作業完成後把檔案版本控制：git commit -am
4.   再把 branch 推到 GitHub  git push origin *( 版本 )*
5.   助教改完會把 branch 合併到 master
6.   **需要 git  pull 手動抓取 GitHub 的資料到本機讓資料同步** ：git pull
